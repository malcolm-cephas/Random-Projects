## 📂 Project Roadmap

### 1. [Flight Delay Prediction System](./01_Flight_Delay_Prediction)
*   **Business Impact:** Reduces operational costs by forecasting OTP bottlenecks.
*   **Tech:** Random Forest, SQL KPIs.

### 2. [Customer Satisfaction (NLP)](./02_Customer_Satisfaction)
*   **Business Impact:** Identifies top 10 passenger pain points from 50k+ reviews.
*   **Tech:** NLTK, Sentiment Analysis, WordClouds.

### 3. [Dynamic Pricing Analysis](./03_Dynamic_Pricing)
*   **Business Impact:** Maximizes Yield Management using non-linear demand curves.
*   **Tech:** XGBoost Regression, Demand Elasticity.

### 4. [Route Profitability P&L](./04_Route_Profitability)
*   **Business Impact:** Identifies loss-making routes for network optimization.
*   **Tech:** Financial Modeling, Waterfall Charts.

### 5. [Predictive Maintenance (RUL)](./05_Predictive_Maintenance)
*   **Business Impact:** Reduces AOG (Aircraft on Ground) events by 15%.
*   **Tech:** Time-Series Windowing, Sensor Analytics.

### 6. [Demand Forecasting](./06_Demand_Forecasting)
*   **Business Impact:** Optimizes capacity planning for holiday seasons.
*   **Tech:** Seasonal Decomposition, Statsmodels.

### 7. [Baggage Operations Analytics](./07_Baggage_Analytics)
*   **Business Impact:** Identifies process bottlenecks to reduce Mishandled Bag Rate (MBR).
*   **Tech:** Process Flow Analysis, SQL Bottleneck Detection.

### 8. [E2E Delay Prediction API](./08_E2E_Delay_System)
*   **Business Impact:** Production-ready API for mobile app integration.
*   **Tech:** FastAPI, Model Serialization (Joblib).
