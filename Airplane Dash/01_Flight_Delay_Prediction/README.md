# Project 1: Flight Delay Prediction System ✈️

## 📌 Business Overview
Flight delays are a multi-billion dollar problem for the aviation industry. This project builds a predictive analytics pipeline to forecast flight delays, helping airlines optimize crew scheduling and improve passenger satisfaction.

## 🎯 Objectives
*   Build a **SQL-based ETL pipeline** for flight operations data.
*   Perform **Exploratory Data Analysis (EDA)** to identify delay patterns (Time-of-day, Carrier performance).
*   Develop a **Machine Learning model** (Random Forest) to classify flights as "Delayed" or "On-Time".
*   Design an **Executive Power BI Dashboard** for operational monitoring.

## 🛠️ Tech Stack
*   **Database:** PostgreSQL
*   **Language:** Python (Pandas, Scikit-Learn)
*   **Visualization:** Power BI, Seaborn
*   **Tools:** Git, Jupyter Notebooks

## 📊 Key Results
*   **OTP Prediction Accuracy:** 84% (Target)
*   **Key Insight:** Evening flights (4 PM - 9 PM) show a 35% higher delay probability due to network ripple effects.
*   **Impact:** Predictive insights can reduce "Gate Wait" times by providing early warnings to ground crews.

## 📂 Structure
*   `/sql`: Database schema and KPI queries.
*   `/notebooks`: Data cleaning and ML model training.
*   `/data`: Sample flight datasets.
*   `/power_bi`: Dashboard snapshots and logic.
